redis-cli各种命令使用示例

Redis版本：4.0
Github：https://github.com/redis/redis/tree/4.0
Redis官方文档（纯英文）：https://redis.io/docs>
Redis官方指令查询（纯英文）：https://redis.io/commands

字符串

get

get

获取值。
当key不存在时返回nil。

set

set

设置值。

getset

getset

设置key的新值，返回key的旧值。
当key不存在时，返回nil。

setex

setex

设置值，并且修改过期时间为 seconds 秒。

setnx

setnx

当key不存在时，为key设置值。

集合

管理命令

module

模块管理命令

  1. load

    module load [args…]

    加载模块

  2. unload

    module unload

    卸载模块

  3. list

    module list

    查看已加载的模块

object

object <refcount|encoding|idletime|freq>

object命令可以检查对象的内部结构

  1. refcount – 返回key的值被引用的次数。
  2. encoding – 返回key的值编码类型。
  3. idletime – 返回key的值的空闲时间，即最后一次访问以来经过的秒数。
  4. freq – 返回访问频率。
